# DIO Bootcamp Java com Spring Boot

## O que é o DIO Bootcamp Java com Spring Boot?

### O Bootcamp Java com Spring Boot é um programa de formação que proporciona aos estudantes o conhecimento para desenvolver aplicações Java com Spring Boot.

## O que você vai aprender?

### Você vai aprender a criar APIs com Spring Boot, além de conhecer a arquitetura de software e o padrão de projeto MVC

## Quem pode participar?

### Você pode participar de qualquer pessoa que tenha interesse em aprender a criar APIs com Spring Boot e conhecer a arquitetura de software e o padrão de projeto MVC
